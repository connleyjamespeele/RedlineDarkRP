# Gambling Module Overview

This folder contains docs for the flips gambling system.

- `sv_flips.lua`: server challenge handling, acceptance, payout, and cancellation.
- `cl_flips.lua`: client flip menu to open and accept matches.

Use `/flips` to open the challenge menu.
Use `/acceptflip <player>` to accept a pending flip from chat.
Use `/flips cancel` to cancel your own challenge and get your money back.
